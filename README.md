# Weather App Project

## Description

A weather app that delivers live forecasts, leveraging geolocation APIs and asynchronous JavaScript to provide users with location-specific weather updates

## Features

**Live Weather Data**: Fetches humidity, wind and conditions.
**Location Search**: Look up current weather conditions for any city.
**Responsive Design**: Fully optimized for desktop, tablet, and mobile screens.

## Tech Stack

**Frontend**: HTML, CSS, JavaScript
**Icons**: Font Awesome
**API**: OpenWeatherMap API

## Installation

Follow these steps to run the project locally on your machine.

1. **Clone the repository:**

   git clone nyuykongmoalvin848-oss/The-Weather-App

2. **Navigate to the project directory:**

   cd WEATHER APP

3. **Set up your API Key:**
   * Sign up at [OpenWeatherMap] to get a free API key.
   * Open script.js  and replace the placeholder text with your actual key
4. **Launch the app:**
   Open index.html and 'GO LIVR' in VS Code.
